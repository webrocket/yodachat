==== INSTRUCTIONS ====

To make YodaChat working you have to prepare your environment:

---> Start Redis server:

     eg. /etc/rc.d/redis start

---> Start WebRocket server:
     sudo webrocket-server
---> Configure vhost for websockets:

     sudo webrocket-admin add_vhost /yoda

---> Copy .env.sample file to .env and fulfill it with your configuration

---> Run `script/boostrap` once again

---> Strart application with foreman:

     foreman start

---> Go to http://localhost:5000 and enjoy!
